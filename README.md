# ProjectTracker

ProjectTracker is an ASP.NET Core 10 web application for managing and tracking projects with budgets, owners, and team members.

## Features

- View and filter projects by maximum budget
- Create new projects with name, description, owner, and budget
- Delete existing projects
- See total budget across all (filtered) projects

## Getting Started

### Prerequisites

- [.NET 10 SDK](https://dotnet.microsoft.com/download)

### Running the application

```bash
dotnet watch
```

The application will be available at `http://localhost:5119`.

### Building

```bash
dotnet build
```
